# db_comp1

Навчальний проєкт з дисципліни "Бази даних".  
Інформаційна система обліку комп’ютерних комплектуючих та замовлень.

## Технології
- Python
- FastAPI
- SQLite

## Запуск

### 1. Клонувати Git репозиторій
```bash
git clone https://github.com/paliichukvladyslav/db_comp1
```

### 2. Виконати скрипт запуску
```bash
./start.sh
```

Після запуску Swagger UI доступний за адресою:
```http://127.0.0.1:8000/docs```
